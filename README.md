# 俞磊的历史地图

这是俞磊制作的历史地图SVG文件


警告！ 所有的地图均只用来示意，并不能用作精确的测绘及定位。图中所有的位置和边界轮廓，不代表任何政治或者宗教的观点，个别边界线涉及现代政治纠纷，本人的地图并无任何政治含义。

技术说明：所有的文件都使用Inkscape的svg格式，虽然一般的svg软件也能编辑，但建议使用Inkscape软件，以避免损失一些附加信息
